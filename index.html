3D Missile Intercept — Proportional Navigation

This project implements a 3D missile-target intercept simulation using Augmented Proportional Navigation (APN) guidance. The simulation models a maneuvering target with sinusoidal acceleration, missile actuator dynamics with saturation limits, and uses RK4 integration for high-fidelity trajectory propagation. The guidance law computes line-of-sight rates and closing velocity to generate lateral acceleration commands that drive the missile to intercept.
